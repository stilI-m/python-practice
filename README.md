# 🚀 Python Practice & Crypto Telegram Bot

Привет! Это мой репозиторий для практики backend-разработки на Python.
Здесь я закрепляю работу с алгоритмами, внешними API и созданием ботов.

## 📌 Главный проект: Crypto Tracker Bot (`bot.py`)

Интерактивный Telegram-бот, который в реальном времени отслеживает курсы криптовалют.

**Что под капотом:**

- **REST API:** Интеграция с публичным API криптобиржи Binance.
- **JSON:** Парсинг и обработка данных от сервера.
- **Telebot:** Асинхронная обработка сообщений пользователя (Event-Driven architecture).
- **Обработка ошибок:** Защита от некорректного ввода (бот отвечает стикером и сообщением об ошибке, если монета не найдена).
- **Веб-сервер (Микросервисы):** Развернут локальный сервер на Flask, который обращается к базе данных и генерирует HTML-страницу со статистикой использования бота.
- **Базы данных:** Спроектирована локальная БД на SQLite. Реализована безопасная запись данных (защита от SQL-инъекций) и чтение истории запросов.
- **Хеширование паролей:** Защита от хакерских атак и возможность просматривать таблицу с данными только админу(авторизация).

## 🛠 Технологии

- Python 3
- `requests`
- `pyTelegramBotAPI` (telebot)
